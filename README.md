🎓 GENAI Academic Help Desk
An AI-powered Academic Help Desk chatbot developed using Python, Flask, LangChain, Hugging Face, ChromaDB, HTML, CSS, and JavaScript. The application answers students' academic queries by retrieving information from PDF documents using Retrieval-Augmented Generation (RAG).

📖 Project Overview
The GENAI Academic Help Desk is an intelligent virtual assistant designed for students. It enables users to ask academic-related questions in natural language and provides accurate responses by retrieving information from PDF documents such as syllabus, placement details, hostel information, library information, and academic circulars.
